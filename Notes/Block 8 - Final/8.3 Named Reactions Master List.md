# 8.3 Named Reactions Master List

## List (to be filled progressively)

| Reaction | Type | What it does |
|----------|------|--------------|
| Markovnikov's Rule | Addition | H adds to C with more H |
| Kharasch (Anti-Markovnikov) | Free radical addition | H adds to C with fewer H (peroxide) |
| Ozonolysis | Oxidative cleavage | C=C → two carbonyl compounds |
| Aldol Condensation | Addition + elimination | Two carbonyls → β-hydroxy carbonyl |
| Cannizzaro Reaction | Disproportionation | Non-enolizable aldehyde → acid + alcohol |
| Tollens Test | Oxidation | Aldehyde → silver mirror |
| Fehling's Test | Oxidation | Aldehyde → brick red ppt |
| Clemmensen Reduction | Reduction | C=O → CH₂ (acidic conditions) |
| Wolff-Kishner Reduction | Reduction | C=O → CH₂ (basic conditions) |
| Fischer Esterification | Condensation | Acid + alcohol → ester |
| Hofmann Degradation | Rearrangement | Amide → 1° amine (one C less) |
| Sandmeyer Reaction | Substitution | Diazonium → Cl/Br/CN compound |
| Reimer-Tiemann | Formylation | Phenol → 2-hydroxybenzaldehyde |
| Kolbe Reaction | Carboxylation | Phenol → salicylic acid |
| Hell-Volhard-Zelinsky | Halogenation | α-halogenation of carboxylic acid |
| Friedel-Crafts | EAS | Alkylation/acylation of benzene |
| Saytzeff's Rule | Elimination | Major product = more substituted alkene |
