# 2.1 Structural Formulas

> [!tip] One Line
> Three ways to write the same compound: complete structural, condensed, and skeletal — you must be able to convert between all three instantly.

---

## Types of Structural Formulas

### 1. Complete Structural Formula (Lewis/Dash)
Shows every atom and every bond explicitly.
```
    H   H   H
    |   |   |
H — C — C — C — H
    |   |   |
    H   H   H
    
    Propane (C₃H₈)
```

Useful for: Understanding bonding, counting electrons, seeing geometry.
Problem: Too slow to draw for large molecules.

---

### 2. Condensed Structural Formula
Groups H atoms with their carbon. Bonds between carbons are implied.
```
CH₃–CH₂–CH₃     or     CH₃CH₂CH₃    (propane)

CH₃–CH(CH₃)–CH₃        (isobutane — branch shown in parentheses)

CH₃–C(CH₃)₂–CH₃        (neopentane — two identical branches, use subscript 2)
```

**Rules for condensed formula:**
- Write C atoms from left to right
- Group H atoms immediately after their C: CH₃, CH₂, CH
- Put branches in parentheses: –CH(CH₃)– means a CH with one CH₃ branch
- If two identical branches: use subscript — C(CH₃)₂ means C with two CH₃ groups

---

### 3. Skeletal Formula (Line-angle formula)
- Each **line end** or **junction** = one carbon
- Hydrogens on carbon are **not shown** (implied)
- Only non-carbon atoms (O, N, Cl...) and their H are shown explicitly

```
Propane:    /\/     (each end and bend = 1 carbon)

But-1-ene:  /\=\   (= shows double bond)

Cyclohexane:  ⬡   (hexagon)
```

**How to read skeletal:**
1. Count line ends and junctions = number of carbons
2. Each carbon needs 4 bonds total — missing bonds are C–H

### Example: Skeletal → Condensed
```
    /\
   /  \
  /    

This is pentane. Count: 5 carbons (2 ends + 3 junctions... actually just count zigzag points)
= CH₃–CH₂–CH₂–CH₂–CH₃
```

---

## Converting Between Formulas — Step by Step

### Condensed → Skeletal
1. Draw each C–C bond as a line/zigzag
2. Erase all H on carbon
3. Keep heteroatoms (O, N, Cl) and H attached to them

**Example:** CH₃–CH₂–OH → skeletal: `/OH` (two carbons as zigzag, -OH shown)

### Skeletal → Condensed
1. Label each junction/end as C
2. Count bonds at each C
3. Remaining bonds (up to 4) are C–H → write as CH₃, CH₂, CH
4. Write out left to right

---

## Molecular Formula from Structural Formula

1. Count all C atoms
2. Count all H atoms
3. Count heteroatoms

**Example:**
```
CH₃–CH(OH)–CH₂–CH₃

C: 4 (the CH₃, CH, CH₂, CH₃)
H: 3+1+2+3 = 9, plus the OH hydrogen = 10
O: 1
Molecular formula: C₄H₁₀O
```

---

## Degree of Unsaturation (DoU) — Important Formula

Also called Index of Hydrogen Deficiency (IHD).

```
DoU = (2C + 2 + N - H - X) / 2

Where: C = carbon count, H = hydrogen count, 
       N = nitrogen count, X = halogen count
       (O and S don't affect the formula)
```

**Interpretation:**
| DoU | Meaning |
|-----|---------|
| 0 | Saturated (no rings, no π bonds) |
| 1 | One ring OR one double bond (C=C or C=O) |
| 2 | Two rings, or two double bonds, or one triple bond |
| 4 | Benzene ring (1 ring + 3 double bonds) |

### Examples
```
C₆H₁₂  →  DoU = (12+2-12)/2 = 1  → one ring OR one C=C
C₆H₆   →  DoU = (12+2-6)/2  = 4  → benzene (confirmed: 1 ring + 3 C=C)
C₄H₈O  →  DoU = (8+2-8)/2   = 1  → could be C=C or C=O (aldehyde/ketone)
C₃H₇Cl →  DoU = (6+2-7-1)/2 = 0  → saturated haloalkane
```

> [!important] Exam Use
> DoU is your first filter when an unknown compound is given. If DoU = 4, almost certainly has benzene ring. If DoU = 0, it's an alkane or haloalkane. This narrows your options immediately.

## Practice Questions

1. Convert to skeletal formula: `CH₃–CH₂–CH(CH₃)–CH₂–CH₃`

2. Write condensed formula for a molecule with skeletal: pentagon shape

3. Calculate DoU for: C₅H₁₀, C₆H₅Cl, C₄H₉N

4. A compound has DoU = 4. What ring system does it likely contain?

5. Write all 3 formulas (complete structural, condensed, skeletal) for but-2-ene.

> [!question]- Answers
> 1. Five-carbon chain with a branch at C3 — draw as zigzag with a branch sticking up at C3
> 2. Cyclopentane: CH₂–CH₂–CH₂–CH₂–CH₂ (ring) = (CH₂)₅
> 3. C₅H₁₀: (10+2-10)/2=1; C₆H₅Cl: (12+2-5-1)/2=4 (benzene!); C₄H₉N: (8+2+1-9)/2=1
> 4. Benzene ring (DoU=4 is the signature of benzene)
> 5. Complete: H-C(H)(H)-C(H)=C(H)-C(H)(H)(H); Condensed: CH₃–CH=CH–CH₃; Skeletal: /=\ (4 carbons with double bond in middle)
