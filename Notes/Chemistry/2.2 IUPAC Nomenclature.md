# 2.2 IUPAC Nomenclature

> [!tip] One Line
> IUPAC name = locant(s) + substituent prefix(es) + parent chain name + suffix (functional group)

## The 6 Rules of IUPAC Naming

### Rule 1: Find the Longest Carbon Chain
The longest continuous chain of carbons = **parent chain** = gives the base name.

```
CH₃–CH(CH₃)–CH₂–CH₃

Longest chain = 4 carbons = butane (not 3, because the branch is not part of the main chain)
The CH₃ branch = methyl substituent
Name: 2-methylbutane
```

> [!warning] Common Mistake
> Students often count the chain through a branch and miss the actual longest chain. Always try multiple paths.

### Rule 2: Number the Chain
Number from the end **closest to the first point of difference** (first substituent/functional group).

```
CH₃–CH(CH₃)–CH₂–CH₂–CH₃

Numbering from left:  2-methylpentane  ✓
Numbering from right: 4-methylpentane  ✗
→ Choose: gives lower locant to substituent → 2 < 4 → left numbering is correct
```

### Rule 3: Name Substituents as Prefixes

| Substituent | Name |
|-------------|------|
| –CH₃ | methyl |
| –C₂H₅ | ethyl |
| –C₃H₇ | propyl |
| –F | fluoro |
| –Cl | chloro |
| –Br | bromo |
| –I | iodo |
| –NO₂ | nitro |
| –NH₂ | amino |
| –OH | hydroxy (when not principal group) |

### Rule 4: Multiple Same Substituents
Use multiplying prefixes: **di** (2), **tri** (3), **tetra** (4), **penta** (5)

```
CH₃–CBr₂–CH₃    →   2,2-dibromopropane
```

### Rule 5: Alphabetical Order of Substituents
When there are different substituents, list them alphabetically.
**Ignore** multiplying prefixes (di, tri) when alphabetizing.

```
3-ethyl-2-methylpentane   (ethyl comes before methyl alphabetically)
NOT: 2-methyl-3-ethylpentane
```

### Rule 6: Functional Group Suffix
The principal functional group gets a suffix, not a prefix.

| Functional Group | Suffix | Example |
|-----------------|--------|---------|
| Alkane (no group) | -ane | butane |
| Alkene (C=C) | -ene | but-1-ene |
| Alkyne (C≡C) | -yne | but-1-yne |
| Alcohol (–OH) | -ol | butan-1-ol |
| Aldehyde (–CHO) | -al | butanal |
| Ketone (C=O) | -one | butan-2-one |
| Carboxylic acid (–COOH) | -oic acid | butanoic acid |
| Amine (–NH₂) | -amine | butylamine / butan-1-amine |

> [!important] Priority Order (highest gets the suffix)
> –COOH > –CHO > C=O (ketone) > –OH > –NH₂ > C=C > C≡C > –X
> 
> Lower priority groups become prefixes: e.g., if –COOH is present, –OH becomes "hydroxy-"

## Step-by-Step Naming Process

**Example:** Name this compound:
```
CH₃–CH(Br)–CH₂–CH(CH₃)–CH₂–CH₃
```

**Step 1:** Find longest chain
Trace the longest path: 6 carbons → **hexane** base

**Step 2:** Number the chain (substituents at lowest locants)
- From left: Br at C2, CH₃ at C4 → locant set {2,4}
- From right: Br at C5, CH₃ at C3 → locant set {3,5}
- {2,4} < {3,5} → number from left

**Step 3:** Identify substituents
- C2: –Br → bromo
- C4: –CH₃ → methyl

**Step 4:** Alphabetical order
bromo (b) comes before methyl (m)

**Final name: 2-bromo-4-methylhexane**

## Naming Alkenes and Alkynes

For double/triple bonds, number the chain so the multiple bond gets the **lowest possible locant**.

```
CH₂=CH–CH₂–CH₃      but-1-ene    (double bond at C1)
CH₃–CH=CH–CH₃       but-2-ene    (double bond at C2)
HC≡C–CH₂–CH₃        but-1-yne
CH₃–C≡C–CH₃         but-2-yne
```

> [!note] New IUPAC Rule
> The locant for the double bond is placed **before** the suffix:
> but-1-ene (not but-1-ene was old rule — now locant directly precedes -ene)
> Both formats accepted in boards.

---

## Naming Cyclic Compounds

Add **cyclo-** before the chain prefix.
```
3 carbons in ring  → cyclopropane
6 carbons in ring  → cyclohexane
6 carbons in ring with 1 double bond → cyclohexene
```

Numbering starts at the carbon with the substituent and goes in the direction that gives lower locants.

## Practice Questions

1. Name: `CH₃–CH₂–CH(CH₃)–CH₂–CH₂–CH₃`

2. Name: `CH₂=CH–CH₂–CH₃`

3. Name: `CH₃–CHCl–CH₂–CH(CH₃)–CH₃`

4. Write structure for: 3-ethyl-2-methylpentane

5. Name: `CH₃–C≡C–CH₂–CH₃`

6. What is wrong with the name "3-methylbutane"? What should it be?

> [!question]- Answers
> 1. 3-methylhexane
> 2. but-1-ene
> 3. 2-chloro-4-methylpentane
> 4. 5C chain, methyl at C2, ethyl at C3: CH₃–CH(CH₃)–CH(C₂H₅)–CH₂–CH₃
> 5. pent-2-yne
> 6. Should be 2-methylbutane. The longest chain has 4 carbons, and the methyl is at C2 (not C3 — number from closer end). "3-methylbutane" implies the branch is at C3, but renumbering from the other end gives C2.
