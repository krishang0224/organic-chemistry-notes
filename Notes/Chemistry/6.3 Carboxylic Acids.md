# 6.3 Carboxylic Acids

> [!tip] One Line
> –COOH group: most acidic common organic acid; can form esters, amides, anhydrides; reduced to alcohols.

---

## Acidity

–COOH donates H⁺; carboxylate ion (–COO⁻) stabilized by resonance over two oxygens.

**Acidity order:**
- HCOOH > CH₃COOH > C₂H₅COOH (electron-donating alkyl groups destabilize carboxylate)
- Cl₃CCOOH > Cl₂CHCOOH > ClCH₂COOH > CH₃COOH (–I groups stabilize carboxylate)
- p-nitrobenzoic acid > benzoic acid > p-methylbenzoic acid

---

## Key Reactions

### 1. Formation of Salts
```
RCOOH + NaOH → RCOONa + H₂O
RCOOH + Na₂CO₃ → RCOONa + CO₂ + H₂O
```
CO₂ evolved with carbonates/bicarbonates — distinguishes COOH from phenol (phenol doesn't react with Na₂CO₃).

### 2. Esterification (Fischer)
```
RCOOH + R'OH ⇌ RCOOR' + H₂O   (conc. H₂SO₄, heat, reversible)
```
Mechanism: acid-catalyzed nucleophilic acyl substitution.

### 3. Acid Chloride Formation
```
RCOOH + PCl₅ → RCOCl + POCl₃ + HCl
RCOOH + SOCl₂ → RCOCl + SO₂ + HCl   (preferred — gases escape, purification easy)
```

### 4. Anhydride Formation
```
2RCOOH → (RCOO)₂O + H₂O   (P₂O₅, heat)
```

### 5. Decarboxylation
```
RCOONa + NaOH → R–H + Na₂CO₃   (CaO, heat — soda lime)
```
HCOOH → CO₂ + H₂ (with H₂SO₄, used to make CO in lab... wait — actually HCOOH → CO + H₂O with conc. H₂SO₄)

### 6. Reduction
```
RCOOH → RCH₂OH   (LiAlH₄ — cannot be done by H₂/Ni)
```

### 7. Hell-Volhard-Zelinsky (HVZ) Reaction
```
RCOOH + Br₂/PBr₃ → RCH(Br)COOH   (α-haloacid)
```
Br substituted at α-carbon. PBr₃ converts COOH to acid bromide first.

## Important Compounds

| Compound | Formula | Notes |
|----------|---------|-------|
| Formic acid | HCOOH | Reduces Tollens' (has –CHO character), found in ants |
| Acetic acid | CH₃COOH | Vinegar; glacial at 16.6°C |
| Oxalic acid | HOOC–COOH | Strong; reduces KMnO₄; two COOH |
| Benzoic acid | C₆H₅COOH | Food preservative; synthesized from toluene oxidation |

## Practice Questions
1. Which is more acidic: CH₃COOH or CCl₃COOH? Why?
2. How is acetic acid converted to methane?
3. Why does formic acid reduce Tollens' reagent but acetic acid doesn't?
4. What is formed when acetic acid reacts with PCl₅?

> [!question]- Answers
> 1. CCl₃COOH — three Cl atoms withdraw electrons via –I effect, stabilizing carboxylate anion more
> 2. CH₃COONa + NaOH → CH₄ + Na₂CO₃ (dry distillation with soda lime / decarboxylation)
> 3. Formic acid has structure H–COOH (has C–H bond like aldehyde) → can be oxidized further; acetic acid (CH₃COOH) has no such C–H on carbonyl
> 4. CH₃COCl (acetyl chloride) + POCl₃ + HCl
