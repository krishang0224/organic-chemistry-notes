# OR1 Named Reactions — Complete List

> [!tip] One Line
> 25 named reactions. Know: reagent → product → mechanism type. These are direct 2-mark gifts in every exam.

---

## Halogenation / Substitution

### 1. Sandmeyer Reaction
ArN₂⁺ + CuCl/CuBr/CuCN → ArCl / ArBr / ArCN
- Replaces diazonium with halide via Cu catalyst
- ArI: use KI (no Cu needed). ArF: Balz-Schiemann (HBF₄ → heat)

### 2. Balz-Schiemann Reaction
ArN₂⁺Cl⁻ + HBF₄ → ArN₂⁺BF₄⁻ → (heat) → ArF + N₂ + BF₃
- Only way to put F directly on benzene ring

### 3. Hell-Volhard-Zelinsky (HVZ)
RCOOH + Br₂/PBr₃ → RCH(Br)COOH (α-bromoacid)
- α-halogenation of carboxylic acid; PBr₃ activates carbonyl

### 4. Appel Reaction (bonus)
ROH + CCl₄/PPh₃ → RCl — converts alcohol to chloride with inversion

## Oxidation Reactions

### 5. Tollens' Test
RCHO + 2[Ag(NH₃)₂]⁺ → RCOO⁻ + 2Ag↓ (silver mirror)
- Aldehydes only (formic acid also works — has CHO character)

### 6. Fehling's / Benedict's Test
RCHO + 2Cu²⁺ → RCOO⁻ + Cu₂O↓ (brick-red)
- Aldehydes only (except aromatic aldehydes don't react with Fehling's)
- Distinguishes aldehyde from ketone

### 7. Baeyer's Test
Alkene/alkyne + cold dil. KMnO₄ → diol (green → brown/purple decolorized)
- Tests for unsaturation; alkenes decolorize, alkanes don't

### 8. Ozonolysis
Alkene + O₃ → ozonide → (Zn/H₂O) → aldehydes/ketones
- Oxidative work-up (H₂O₂): aldehydes → carboxylic acids
- Reductive work-up (Zn/H₂O): aldehydes stay

## Reduction Reactions

### 9. Clemmensen Reduction
C=O → CH₂ using Zn-Hg / conc. HCl (acidic medium)
- Converts ketone/aldehyde to alkane (removes oxygen completely)
- Use when molecule is acid-sensitive

### 10. Wolff-Kishner Reduction
C=O → CH₂ using NH₂NH₂ + KOH / ethylene glycol (basic medium)
- Same result as Clemmensen, but basic conditions
- Use when molecule is base-sensitive... wait — use when acid-sensitive ← Clemmensen, when base-sensitive → prefer alternative

### 11. Rosenmund Reduction
RCOCl + H₂/Pd-BaSO₄ → RCHO (aldehyde)
- Acid chloride → aldehyde (catalyst poisoned to stop at aldehyde stage)

### 12. Stephen Reduction
RCN + SnCl₂/HCl → RCHO
- Nitrile → aldehyde (via imine intermediate)

---

## Addition Reactions

### 13. Markovnikov's Rule
H–X adds to alkene: H goes to carbon with more H's (more stable carbocation intermediate)
- Anti-Markovnikov with peroxide (HBr only, free radical mechanism)

### 14. Aldol Condensation
2RCHO + dilute NaOH → β-hydroxyaldehyde → (heat) → α,β-unsaturated aldehyde + H₂O
- Requires α-hydrogen; acetone undergoes crossed-aldol with HCHO

### 15. Cannizzaro Reaction
2HCHO + conc. NaOH → HCOOH + CH₃OH (disproportionation)
- Only aldehydes with NO α-H (HCHO, PhCHO, CCl₃CHO)

### 16. Perkin Reaction
ArCHO + (RCO)₂O + RCOONa → ArCH=CR–COOH (cinnamic acid derivative)
- Condensation of aromatic aldehyde with acid anhydride

---

## Aromatic Reactions

### 17. Reimer-Tiemann Reaction
Phenol + CHCl₃ + NaOH → salicylaldehyde (2-hydroxybenzaldehyde)
- Electrophile: dichlorocarbene :CCl₂ (generated from CHCl₃/NaOH)
- Formylation of phenol ring at ortho position

### 18. Kolbe Reaction (Kolbe-Schmitt)
Sodium phenoxide + CO₂ (pressure) + NaOH → sodium salicylate → H⁺ → salicylic acid
- Carboxylation of phenol ring at ortho position

### 19. Friedel-Crafts Alkylation
ArH + RCl + AlCl₃ → Ar–R + HCl
- Limitation: rearrangement, polyalkylation

### 20. Friedel-Crafts Acylation
ArH + RCOCl + AlCl₃ → Ar–COR + HCl
- No rearrangement (acylium ion stable)

---

## N-Based Reactions

### 21. Gabriel Synthesis
Phthalimide + KOH + RX → N-alkylphthalimide + H₂/hydrazine → RNH₂ (1° amine only)
- Gives pure primary amines; no 2° or 3° contamination

### 22. Hofmann Bromamide Degradation
RCONH₂ + Br₂ + 4NaOH → RNH₂ + Na₂CO₃ + 2NaBr
- Amide → amine; loses ONE carbon (degradation)

### 23. Carbylamine Reaction
RNH₂ + CHCl₃ + 3KOH → R–NC (isocyanide) — foul smell
- Test for primary amines ONLY (1° aliphatic or aromatic)

---

## Misc

### 24. Iodoform Reaction
CH₃COR (or CH₃CHOH–R) + I₂/NaOH → CHI₃↓ (yellow) + RCOONa
- Positive: CH₃COCH₃, CH₃CHO, C₂H₅OH, CH₃CH(OH)R

### 25. Victor Meyer Test
Distinguishes 1°/2°/3° alcohols by colour with NO₂ group:
- 1°: red; 2°: blue; 3°: colourless
