# 6.2 Aldehydes & Ketones

> [!tip] One Line
> Both have C=O. Aldehydes (at chain end) are more reactive than ketones (mid-chain) and can be oxidized; ketones cannot.

---

## Nucleophilic Addition ‚ÄĒ General Mechanism
C=O carbon is electrophilic (őī+) ‚Üí nucleophile attacks C ‚Üí addition product

```
Nu:‚ĀĽ + C=O ‚Üí Nu‚ÄďC‚ÄďO‚ĀĽ ‚Üí (protonation) ‚Üí Nu‚ÄďC‚ÄďOH
```

Aldehydes more reactive than ketones (less steric, more electrophilic C=O)

## Key Reactions

### 1. Addition of HCN ‚Üí Cyanohydrin
```
RCHO + HCN ‚Üí RCH(OH)CN   (cyanohydrin)
```
Used to increase carbon chain length.

### 2. Addition of NaHSO‚āÉ ‚Üí Bisulphite adduct
```
RCHO + NaHSO‚āÉ ‚Üí RCH(OH)SO‚āÉNa   (white crystalline solid)
```
Ketones react only if small (methyl ketones). Used for purification.

### 3. Reaction with NH‚āā‚ÄďG (nitrogen nucleophiles)
| Reagent | Product | Name |
|---------|---------|------|
| NH‚āāOH (hydroxylamine) | R‚ÄďCH=N‚ÄďOH | Oxime |
| NH‚āāNH‚āā (hydrazine) | R‚ÄďCH=N‚ÄďNH‚āā | Hydrazone |
| NH‚āā‚ÄďNH‚ÄďC‚āÜH‚āÖ | R‚ÄďCH=N‚ÄďNHC‚āÜH‚āÖ | Phenylhydrazone |
| NH‚āā‚ÄďNH‚ÄďCO‚ÄďNH‚āā | R‚ÄďCH=N‚ÄďNHCONH‚āā | Semicarbazone |

These are used to identify and characterize carbonyl compounds (solid derivatives).

### 4. Aldol Condensation (base, dilute NaOH)
```
2CH‚āÉCHO ‚Üí CH‚āÉCH(OH)CH‚āāCHO   (3-hydroxybutanal, aldol product)
‚Üí (heat) ‚Üí CH‚āÉCH=CHCHO + H‚āāO  (crotonaldehyde, aldol condensation)
```
Requires őĪ-hydrogen. Acetone undergoes cross-aldol.

### 5. Cannizzaro Reaction (conc. NaOH, no őĪ-H)
```
2HCHO ‚Üí HCOOH + CH‚āÉOH   (formic acid + methanol)
2C‚āÜH‚āÖCHO ‚Üí C‚āÜH‚āÖCOOH + C‚āÜH‚āÖCH‚āāOH
```
Disproportionation ‚ÄĒ one aldehyde oxidized, other reduced. Only aldehydes with NO őĪ-H.

### 6. Oxidation Tests (to distinguish aldehyde from ketone)

**Tollens' Test (AgNO‚āÉ/NH‚āÉ):**
```
RCHO + 2[Ag(NH‚āÉ)‚āā]‚Āļ + OH‚ĀĽ ‚Üí RCOO‚ĀĽ + 2Ag‚Üď + 4NH‚āÉ + H‚āāO
```

**Fehling's/Benedict's Test:**
```
RCHO + 2Cu¬≤‚Āļ ‚Üí RCOO‚ĀĽ + Cu‚āāO‚Üď  (brick-red ppt)
```
Brick-red precipitate. Ketones don't react (except methyl ketones with Fehling's ‚ÄĒ debated).

### 7. Reduction
**Clemmensen Reduction:** C=O ‚Üí CH‚āā (Zn-Hg / conc. HCl, acidic)
**Wolff-Kishner Reduction:** C=O ‚Üí CH‚āā (NH‚āāNH‚āā, KOH, ethylene glycol, basic)

### 8. Iodoform Test (CHI‚āÉ)
Methyl ketones (CH‚āÉ‚ÄďCO‚ÄďR) + I‚āā/NaOH ‚Üí CHI‚āÉ‚Üď (yellow) + RCOONa
Also: ethanol and acetaldehyde give iodoform (they get oxidized to methyl ketone first).

## Practice Questions

> 3. Cannizzaro reaction ‚Üí HCOOH + CH‚āÉOH
