# 3.3 Hyperconjugation

> [!tip] One Line
> Delocalization of σ (C–H) bonding electrons into adjacent empty p orbital or π system — also called "no-bond resonance."

---

## What is It

When a C–H bond is adjacent to an empty p orbital (carbocation) or a π bond, the σ electrons partially overlap and delocalize. This stabilizes the system.

```
     H                H
     |                |
 H — C — C⁺   ↔   H — C = C  +  H⁺
     |                |
     H                H
(normal)         (no-bond structure)
```

The actual structure is a hybrid — the + charge is spread out.

---

## Key Rule
**More C–H bonds adjacent to the + charge (or π bond) = more hyperconjugation = more stable**

This is why alkyl groups stabilize carbocations and alkenes.

---

## Carbocation Stability
```
(CH₃)₃C⁺  >  (CH₃)₂CH⁺  >  CH₃CH₂⁺  >  CH₃⁺
   3°              2°              1°         methyl
(9 C–H)        (6 C–H)         (3 C–H)     (0 C–H)
```
More adjacent C–H bonds = more hyperconjugation = more stable

## Alkene Stability
More substituted alkene = more hyperconjugation = more stable
```
(CH₃)₂C=CH₂  >  CH₃CH=CH₂  >  CH₂=CH₂
  (6 C–H)          (3 C–H)       (0 C–H)
```

## Free Radical Stability
Same order as carbocations: 3° > 2° > 1° > methyl

---

## Connection to Markovnikov's Rule
When HX adds to an alkene, H⁺ adds to the carbon that forms the **more stable carbocation** (more substituted). This is directly explained by hyperconjugation stabilizing the 2° or 3° carbocation over 1°.

---

## Practice Questions

1. Why is (CH₃)₃C⁺ more stable than CH₃CH₂⁺?
2. Arrange in stability: propene, ethene, 2-methylpropene
3. In CH₃CH=CH₂ + HBr, which carbon gets Br? Why?

> [!question]- Answers
> 1. (CH₃)₃C⁺ has 9 adjacent C–H bonds for hyperconjugation vs 3 in CH₃CH₂⁺
> 2. 2-methylpropene > propene > ethene (more substitution = more hyperconjugation)
> 3. C2 gets Br — H adds to C1 (forming 2° carbocation at C2, more stable than 1° at C1)
