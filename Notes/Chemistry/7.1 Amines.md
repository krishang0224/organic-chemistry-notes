# 7.1 Amines

> [!tip] One Line
> Amines = nitrogen with lone pair = basic + nucleophilic. Basicity: aliphatic > NH₃ > aniline (resonance kills lone pair).

---

## Classification
- **1° amine:** R–NH₂ (one R on N)
- **2° amine:** R₂NH (two R on N)
- **3° amine:** R₃N (three R on N)
- **4° ammonium salt:** R₄N⁺X⁻ (no lone pair, not basic)

---

## Basicity

Lone pair on N accepts H⁺ → ammonium ion.

**Aliphatic basicity order (aqueous):**
2° > 1° > 3° > NH₃ (due to combined inductive + solvation effects)

**Aniline vs NH₃:**
Aniline (C₆H₅NH₂) << NH₃ << aliphatic amines
→ Lone pair on N delocalized into benzene ring (resonance) → less available for protonation → very weakly basic (pKb ≈ 9.4)

**Effect of substituents on aniline:**
- Electron-withdrawing (–NO₂ at p or o): LESS basic (further delocalization)
- Electron-donating (–CH₃ at p): MORE basic
- p-nitroaniline < aniline < p-methylaniline

---

## Key Reactions

### 1. Acylation (with acid chloride or anhydride)
```
RNH₂ + RCOCl → RCONHR + HCl   (amide formed)
```
Used to protect –NH₂ in aromatic synthesis (amide is weaker o/p director, easier to remove later).

### 2. Reaction with Nitrous Acid (NaNO₂ + HCl)

**Primary aliphatic amine:**
```
RNH₂ + HNO₂ → ROH + N₂↑ + H₂O   (very reactive diazonium — immediately decomposes)
```
N₂ gas evolved — used as a test for 1° aliphatic amine.

**Primary aromatic amine (0–5°C):**
```
ArNH₂ + HNO₂ → Ar–N₂⁺Cl⁻   (diazonium salt — stable, useful!)
```
Must be kept cold or decomposes.

**Secondary amine:**
```
R₂NH + HNO₂ → R₂N–NO   (N-nitrosamine, yellow oil)
```

**Tertiary amine:** no reaction (or salt formation with acid).

### 3. Diazonium Salt Reactions (ArN₂⁺)

| Reaction | Product |
|---------|---------|
| + H₂O/H⁺ (heat) | ArOH (phenol) |
| + CuCl (Sandmeyer) | ArCl |
| + CuBr (Sandmeyer) | ArBr |
| + CuCN (Sandmeyer) | ArCN |
| + KI | ArI (no copper needed) |
| + HBF₄ | ArF (Balz-Schiemann) |
| + C₆H₅OH / alkaline (coupling) | Azo dye (–N=N–) |

### 4. Carbylamine Reaction (test for 1° amines only)
```
RNH₂ + CHCl₃ + 3KOH → R–NC + 3KCl + 3H₂O   (isocyanide/carbylamine — foul smell)
```
Only 1° amines (aliphatic or aromatic) give this test.

### 5. Hofmann Bromamide Degradation
```
RCONH₂ + Br₂ + 4NaOH → RNH₂ + Na₂CO₃ + 2NaBr + 2H₂O
```
Amide → amine (loses one carbon). Used to decrease chain length.

---

## Preparation of Amines
- **Gabriel synthesis:** phthalimide + KOH + R–X → 1° amine (only 1°, no 2° or 3°)
- **Reduction of nitro compound:** ArNO₂ + 6[H] → ArNH₂ (Fe/HCl or Sn/HCl)
- **Reduction of nitrile:** RCN + 4[H] → RCH₂NH₂ (LiAlH₄)
- **Hofmann degradation (above)**

---

## Practice Questions
1. Arrange in basicity: CH₃NH₂, (CH₃)₂NH, C₆H₅NH₂, NH₃
2. What happens when aniline reacts with NaNO₂/HCl at 0°C?
3. How is aniline converted to fluorobenzene?
4. Which test distinguishes 1° from 2° and 3° amines?

> [!question]- Answers
> 1. (CH₃)₂NH > CH₃NH₂ > NH₃ > C₆H₅NH₂
> 2. Benzene diazonium chloride (C₆H₅N₂⁺Cl⁻) formed — stable at 0–5°C
> 3. Diazotization (NaNO₂/HCl, 0°C) → C₆H₅N₂⁺BF₄⁻ (HBF₄) → heat → C₆H₅F (Balz-Schiemann)
> 4. Carbylamine test — only 1° amines give foul-smelling isocyanide with CHCl₃/KOH
