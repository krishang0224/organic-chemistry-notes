> [!tip] One Line

## 1. Free Radical Halogenation (Alkanes)

```
Initiation:   Cl₂ → 2Cl•        (hν or heat)
Propagation:  Cl• + CH₄ → HCl + CH₃•
              CH₃• + Cl₂ → CH₃Cl + Cl•
Termination:  Cl• + Cl• → Cl₂  (or CH₃• + Cl• → CH₃Cl)
```

**Selectivity:** 3° > 2° > 1° (stability of radical intermediate)
Bromination more selective than chlorination (Br radical is less reactive, more choosy).

**Rate ratio (monohalogenation):**
- Chlorination: 3°:2°:1° = 5:4:1
- Bromination: 3°:2°:1° = 1600:82:1

## 2. Electrophilic Addition to Alkenes

### Mechanism (HBr addition)
```
Step 1: H⁺ attacks π bond → carbocation (rate-determining)
Step 2: Br⁻ attacks carbocation → product
```
Markovnikov: H⁺ adds to less substituted C → more stable carbocation forms.

### Anti-Addition (Br₂/CCl₄)
```
Br₂ → forms bromonium ion bridge
Nu attacks from opposite face → trans (anti) product
```
→ **Trans** product; proves mechanism involves cyclic bromonium ion.

### Ozonolysis Mechanism
```
O₃ → [1,3-dipolar cycloaddition] → molozonide → rearrangement → ozonide
Zn/H₂O (reductive) → aldehydes/ketones
H₂O₂ (oxidative) → carboxylic acids (aldehydes oxidized further)
```

## 3. SN1 Mechanism (Full)

```
Step 1: R–X → R⁺ + X⁻         (slow, rate-determining — ionization)
Step 2: R⁺ + Nu:⁻ → R–Nu      (fast — from both faces)
```

**Rate = k[RX]** — first order

**Stereochemistry:** Racemization (planar carbocation attacked equally from both faces)
(In practice: slight excess of inversion because solvent partially blocks front face)

**Carbocation stability:** 3° > 2° > 1° > CH₃⁺
**Rearrangements possible:** hydride or alkyl shift to form more stable carbocation

**Favored by:**
- 3° substrate (stable carbocation)
- Polar protic solvent (stabilizes ions: H₂O, EtOH, AcOH)
- Weak nucleophile

---

## 4. SN2 Mechanism (Full)

```
Nu:⁻ ----C----X → [Nu---C---X]‡ → Nu–C + X⁻
(backside, 180°)   (transition state)
```

**Rate = k[RX][Nu]** — second order

**Stereochemistry:** Complete inversion (Walden inversion)

**Favored by:**
- 1° substrate (least steric hindrance)
- Polar aprotic solvent (DMSO, acetone, DMF — doesn't solvate Nu, keeps it reactive)
- Strong nucleophile (I⁻ > Br⁻ > Cl⁻ > F⁻ in aprotic; reversed in protic)

**Cannot happen at 3° carbon** (too hindered for backside attack)

---

## 5. E2 Mechanism

```
B:⁻ + H–C–C–X → B–H + C=C + X⁻   (one step, concerted)
```

**Rate = k[RX][B]**

**Stereochemistry:** Anti-periplanar (H and X must be 180° apart — confirmed by Newman projection)
→ **trans** (anti) alkene preferred from anti-periplanar elimination

**Favored by:** Strong bulky base (KO-tBu), high temp, any substrate (works well with 2° and 3°)

**Saytzeff product:** more substituted alkene (thermodynamic stability)
**Hofmann product:** less substituted alkene (with bulky base, kinetic)

## 6. Nucleophilic Addition to Carbonyls (General)

```
Nu:⁻ attacks C=O (δ+) → tetrahedral alkoxide
Protonation → alcohol product
```

**Reactivity order:** HCHO > RCHO > RCOR' (steric + electronic)
- More substituents = more steric hindrance
- Alkyl groups donate electrons (+I) → reduce δ+ on C → less electrophilic

### Cyanohydrin formation:
```
RCHO + HCN ⇌ RCH(OH)CN
(catalyzed by KCN — CN⁻ is actual nucleophile)
```

### Addition of Grignard (R'MgX):
```
HCHO + RMgX → 1° alcohol
RCHO + R'MgX → 2° alcohol
RCOR'' + R'MgX → 3° alcohol
```

## 7. Aldol Condensation Mechanism

```
Step 1: OH⁻ removes α-H → enolate anion
Step 2: Enolate attacks C=O of another molecule → alkoxide
Step 3: Protonation → β-hydroxyaldehyde (aldol product)
Step 4: Heat → –H₂O → α,β-unsaturated aldehyde (condensation)
```

**Key:** Both aldol product AND condensation product are asked.
- Aldol = β-hydroxy compound (no heat)
- Condensation = α,β-unsaturated (with heat)

## 8. Electrophilic Aromatic Substitution (EAS) Mechanism

```
Step 1: E⁺ + π cloud → arenium ion (σ-complex, sp3 carbon, non-aromatic)
Step 2: Loss of H⁺ (base removes) → rearomatization
```

**Why substitution not addition?** Addition would destroy aromaticity (–150 kJ/mol resonance energy = too costly).

**Nitration electrophile:** NO₂⁺ generated by H₂SO₄ + HNO₃ → HSO₄⁻ + H₂NO₃⁺ → NO₂⁺ + H₂O

**Halogenation electrophile:** Cl⁺ generated by Cl₂ + FeCl₃ → FeCl₄⁻ + Cl⁺

## 9. Diazotization & Coupling

```
ArNH₂ + NaNO₂ + HCl (0–5°C) → ArN₂⁺Cl⁻   (diazonium salt)
ArN₂⁺ + C₆H₅OH (alkaline) → Ar–N=N–C₆H₄–OH  (azo dye, orange-red)
ArN₂⁺ + ArNH₂ (slightly acidic) → Ar–N=N–Ar' (azo dye)
```

Coupling is EAS — diazonium is weak electrophile, needs activated ring (–OH, –NR₂).

---

## 10. Esterification Mechanism (Fischer)

```
Step 1: Protonation of C=O by H⁺
Step 2: Nucleophilic attack by R'OH
Step 3: Proton transfer
Step 4: Loss of H₂O
Step 5: Deprotonation → ester
```

**Reversible** → use excess alcohol or remove water to drive forward (Le Chatelier).

## Key Intermediates — Don't Confuse

| Intermediate | Structure | Forms in |
|-------------|-----------|---------|
| Carbocation | R⁺ (planar, sp2) | SN1, E1 |
| Carbanion | R⁻ (sp3) | Grignard, organolithium |
| Free radical | R• (sp2, planar) | Free radical halogenation |
| Enolate | –CH=C–O⁻ | Aldol, Claisen |
| Bromonium ion | cyclic Br⁺ bridge | Br₂ addition to alkene |
| Arenium ion (σ-complex) | cyclohexadienyl cation | EAS |
| Carbene :CR₂ | neutral, 6e⁻ | Reimer-Tiemann (dichlorocarbene) |
