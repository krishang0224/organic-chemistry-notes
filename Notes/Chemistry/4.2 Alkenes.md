# 4.2 Alkenes

> [!tip] One Line
> Unsaturated hydrocarbons with C=C. Reactive π bond makes them undergo electrophilic addition reactions.

---

## Key Reactions

### 1. Hydrogenation (H₂ / Ni, Pt, or Pd catalyst)
```
CH₂=CH₂ + H₂ → CH₃–CH₃
```
Syn addition (both H on same side). Used to make margarine from oils.

### 2. Addition of HX (Hydrohalogenation)
```
CH₃–CH=CH₂ + HBr → CH₃–CHBr–CH₃   (major, Markovnikov)
                  → CH₃–CH₂–CH₂Br  (minor)
```
**Markovnikov's Rule:** H adds to the carbon with more H's (forms more stable carbocation)

**Mechanism:** H⁺ adds first → carbocation → X⁻ attacks

**Anti-Markovnikov (Peroxide/Kharasch effect):**
```
CH₃–CH=CH₂ + HBr → CH₃–CH₂–CH₂Br  (free radical, peroxide present)
```
Free radical mechanism — Br• adds first to less substituted C

> [!important] Only HBr shows anti-Markovnikov with peroxides. NOT HCl or HI.

### 3. Addition of H₂O (Hydration) — H₃PO₄ catalyst
```
CH₂=CH₂ + H₂O → CH₃CH₂OH   (follows Markovnikov)
```

### 4. Addition of X₂ (Halogenation)
```
CH₂=CH₂ + Br₂ → CH₂Br–CH₂Br   (1,2-dibromoethane)
```
Anti addition (X adds to opposite faces). Bromine water test — decolourisation of reddish-brown Br₂.

### 5. Addition of HOX (Halohydrin formation)
```
CH₂=CH₂ + HOCl → CH₂OH–CH₂Cl
```
OH goes to more substituted C (Markovnikov), X to less substituted.

### 6. Ozonolysis
```
R–CH=CH–R' + O₃ → R–CHO + R'–CHO   (reductive, Zn/H₂O)
R–CH=CH–R' + O₃ → R–COOH + R'–COOH (oxidative, H₂O₂)
```
Cleaves the double bond. Used to identify structure of unknown alkenes.

### 7. Oxidation with KMnO₄
**Cold, dilute KMnO₄ (Baeyer's reagent):**
```
CH₂=CH₂ → HOCH₂–CH₂OH   (diol, syn addition)
```
Purple KMnO₄ decolourises — test for unsaturation.

**Hot, conc. KMnO₄:**
Cleaves double bond → carboxylic acids (or CO₂ if terminal)

## Practice Questions
1. Major product: propene + HBr (no peroxide)?
2. Major product: propene + HBr (with peroxide)?
3. What does decolourisation of bromine water indicate?
4. Ozonolysis of 2-butene gives what products (reductive)?
5. Why does only HBr (not HCl) show anti-Markovnikov addition with peroxides?

> [!question]- Answers
> 1. 2-bromopropane (Markovnikov)
> 2. 1-bromopropane (anti-Markovnikov, free radical)
> 3. Presence of C=C (unsaturation) in the compound
> 4. 2 molecules of CH₃CHO (ethanal)
> 5. HCl: Cl• addition is endothermic (not feasible); HI: I• too stable, doesn't add. Only Br• has right energy balance.
