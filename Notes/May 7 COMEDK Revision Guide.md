## Maths — Last Day Checklist

### Must-Verify
- [ ] Integration by parts: ILATE order
- [ ] King's property: ∫₀ᵃ f(x)dx = ∫₀ᵃ f(a–x)dx
- [ ] Wallis: ∫₀^(π/2) sin^n x = ∫₀^(π/2) cos^n x
- [ ] Area between curves: ∫[upper – lower]dx, split at crossings
- [ ] Linear ODE: IF = e^∫P dx; solution: y·IF = ∫Q·IF dx
- [ ] Binomial middle term: T_(n/2+1) for even n
- [ ] Conic tangency condition: for y=mx+c tangent to y²=4ax → c = a/m
- [ ] Ellipse: SP+S'P = 2a; focal chord semi-latus = b²/a

### Tricky Traps to Avoid
1. **Inverse trig domain:** sin⁻¹x defined only for x ∈ [–1,1]
2. **Log:** log of negative = undefined; base ≠ 1
3. **Determinant:** expand along row/col with most zeros
4. **Probability:** mutually exclusive ≠ independent
5. **Circle tangent vs chord of contact:** same equation form — read question carefully
