## VIT High-Yield Topics (Don't Skip These)

### Chemistry
- [ ] Named reactions (minimum 20: Markovnikov, Aldol, Cannizzaro, Tollens, Fehling, SN1/SN2, Sandmeyer, Reimer-Tiemann, Kolbe, HVZ, Clemmensen, Wolff-Kishner)
- [ ] Functional group tests (Lucas, Tollens, Fehling, Victor Meyer, Hinsberg)
- [ ] Coordination compounds: IUPAC, EAN rule, isomerism
- [ ] Electrochemistry: Nernst equation, cell EMF
- [ ] Chemical equilibrium: Kp, Kc, Le Chatelier
- [ ] p-block: key reactions of N, P, S, Cl compounds

### Physics
- [ ] Kinematics formulas
- [ ] Electrostatics: Coulomb, Gauss law, capacitors
- [ ] Current electricity: Kirchhoff, Wheatstone
- [ ] Modern physics: photoelectric effect, Bohr model
- [ ] Optics: mirror/lens formula, diffraction/interference basics

### Maths
- [ ] Definite integration standard results
- [ ] Matrices inverse + Cayley-Hamilton
- [ ] Probability: Bayes theorem, binomial distribution
- [ ] 3D: distance formulas, angle between planes/lines
- [ ] Limits: standard forms, L'Hôpital
